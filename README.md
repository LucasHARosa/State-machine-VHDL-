# State machine VHDL 
Uma máquina de estado se fundamenta, como o próprio nome diz, em direcionar o funcionamento de um software em um número finito de estados, sendo cada um desses estados uma situação relevante do sistema.
## O Projeto:
O projeto e um contador que varia de 0 a 59, simulando um cronômetro de 1 minuto. Nosso projeto conta com 3 botoes:
* Reset: deixa o contador em 0
* Enable: ativa a contagem
* Up/Down: define o modo da contagem, podendo ser de 0 a 59 ou de 59 a 0
